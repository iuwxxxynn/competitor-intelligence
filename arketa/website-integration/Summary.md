# Website Integration - Arketa

**URL:** https://dashboard.arketa.com/dashboard/sales-channels/website-integration
**Page Type:** Sales Channels / Website Widget Configuration
**Crawl Date:** 2026-05-08

## Purpose
Configure and embed the Arketa booking widget into external websites. Allows studios to sell classes and memberships directly from their own site.

## Style Settings
- Button Color: #263238 (dark teal/charcoal default)
- Background Color: #ffffff (white)
- Text Color: #343a40 (dark gray)
- Font: Configurable to match studio branding

## CTAs and Buttons
- Copy Embed Code: Copies iframe or JS snippet for website embedding
- Save Style Settings: Applies color/font changes to the widget
- Preview Widget: Live preview of widget appearance

## UX Observations
- Color picker inputs allow precise brand color matching for button, background, and text
- Embed code copy is a single-click action reducing friction for non-technical studio owners
- Preview functionality reduces guesswork during widget configuration
- Placement under Sales Channels frames this as revenue-generating, not just operational

## Business Observations
- Website widget is a critical conversion tool for studios to monetize their own web traffic
- Style customization positions Arketa as a white-label-capable platform
- Sales Channels section suggests future additions (e.g., app marketplace, POS)

## Operational Observations
- Studios without developers can embed via simple copy-paste
- Brand color matching reduces need for custom CSS overrides post-embed
- Widget updates likely propagate automatically without re-embedding
