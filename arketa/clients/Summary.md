# Arketa - Clients (CRM)

**URL:** https://dashboard.arketa.com/dashboard/customers/clients
**Captured:** 2026-05-08

  ## Purpose
  Client database and CRM. Search, filter, and manage all studio clients.

  ## Columns
  Name | Email | Phone | Tags | Last Interaction

## Filters
- Search for a client
- Tags filter
- Display (column visibility)

  ## Primary CTAs
  - **More** dropdown: Upload CSV, Export CSV, Create segment, Create custom field
- **Add new**: Create individual client

## Default Data
- Walk-in Client (system auto-created for anonymous POS)

  ## UX Observations
  - Tags and Last Interaction column drive recency-based outreach
- CSV import enables bulk client migration
- Segments link CRM directly to Marketing workflows
