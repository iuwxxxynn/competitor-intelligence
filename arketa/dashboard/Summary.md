# Arketa - Dashboard (Home)

**URL:** https://dashboard.arketa.com/dashboard
**Captured:** 2026-05-08

## Purpose
Main home dashboard for studio owners. Central hub for daily operations monitoring, quick actions, and KPI visibility.

## Primary CTAs
- Customize Dashboard
- Connect (Stripe payments)
- Go to schedule
- New Sale (pinned in sidebar)

## Widgets
1. Notification Center (tabs: All | Failed Subscriptions | Class Packs Low | Class Packs Expiring | Tasks)
2. Upcoming Schedule
3. Favorite Reports (Total Sales, Active Memberships, Total Visits - Last 7 days)

## UX Observations
- Sidebar icon-only when collapsed, labeled when expanded
- New Sale pinned at top of sidebar for fast POS access
- Notification Center uses tabbed alert categories
- Home dashboard is fully customizable (drag-and-drop widgets)
- No onboarding checklist visible - users must self-discover

## Key Differentiators vs Competitors
- Customizable home dashboard (Mindbody has fixed layout)
- Notification Center for proactive churn prevention
- Refer and earn $50 referral program in sidebar
